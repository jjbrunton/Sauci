# Sauci Mobile Release Notes

## v1.0.1 (Build 30) - January 2025

### New Features
- **Match Archiving** - Archive old matches to keep your active list focused
- **Daily Swipe Limits** - Pace your discovery with daily question limits
- **Intensity Preferences** - Set your comfort level for question intensity (1-5 scale)
- **Infinite Scroll** - Smoother browsing through Matches and Answers with infinite scroll
- **Chat Date Separators** - Easily see when conversations happened
- **Lifecycle Notifications** - Get reminded to check in with your partner
- **Message Reporting** - Report inappropriate messages for review
- **Account Deletion** - Full account deletion option in settings
- **Reset Progress** - Start fresh by resetting your question progress

### Improvements
- **Enhanced Swipe UI** - Smoother swiping experience with visual refinements
- **Improved Chat Layout** - Match card moved into message list for cleaner design
- **Keyboard Behavior** - Chat header collapses when keyboard opens for more screen space
- **Profile Settings** - New appearance and notification preferences
- **Premium Design** - Updated subscription card with gold styling
- **Version Display** - App version now visible in settings

### Bug Fixes
- Fixed subscription status incorrectly showing "Via partner's" after direct purchase
- Fixed Android navigation issues
- Various performance improvements and stability fixes

### Privacy
- Removed partner answered indicator for enhanced privacy
