# New Question Types Draft - Non-Explicit Packs

New questions using **audio**, **photo**, and **who_likely** types for all non-explicit question packs. Questions match the tone, intensity range, and theme of each pack.

**Question types:**
- **audio** - User records a voice response (max 60s)
- **photo** - User uploads/captures a photo
- **who_likely** - "Who is more likely to..." (each partner picks who)

None of these types use partner_text.
